## 核心功能

> 从任务创建到多级分解，从好友协作到实时跟踪，任务分解为您提供完整的工作管理闭环。

| 功能 | 说明 |
|------|------|
| 🎯 **智能任务创建** | 快速创建任务，指定名称、内容、预计完成时间与执行人，支持多人同时承担同一任务目标 |
| 🌿 **多级任务分解** | 任意层级的父子任务结构，将大目标层层拆解为具体可执行的子任务，树状展示一目了然 |
| 🎙️ **语音录入 + AI 整理** | 语音口述任务内容，AI 自动整理成结构化任务列表，支持最长 2 分钟的连续录音识别 |
| 👥 **好友与团队协作** | 添加好友、发送协作请求，将任务分配给团队成员，共同推进项目进度，收到实时状态更新 |
| 🔔 **SSE 实时推送** | 基于 Server-Sent Events 技术，好友请求、任务更新等消息实时推送到您的设备，零延迟感知 |
| 📊 **任务状态全程跟踪** | 五种任务状态：未完成、工作中、已完成、退回、终止，完整记录每项任务的生命周期 |

## AI 超级能力

集成业界领先的 **DeepSeek 大语言模型**，只需用自然语言描述您的工作目标，AI 即可智能生成多层级任务清单，帮助您规划出清晰的执行路径。

支持 **语音录入 → AI 识别 → 结构化整理** 的完整流程，让任务规划效率提升数倍。

**AI 能力亮点：**

- ✦ 自然语言描述 → 自动生成任务树
- ✦ 语音实时转文字，精准识别中文内容
- ✦ AI 整理后可手动编辑，灵活可控
- ✦ 按需调用，节省 Token 成本，系统配置灵活

### AI 工作示例

**您说：**
> 下个月要做年度汇报，需要收集各部门数据、制作 PPT、安排汇报场地和邀请领导...

**AI 整理结果：**

```
✦ 年度汇报准备
  ├ 数据收集阶段
  │   ├ 各部门数据汇总
  │   └ 数据审核与核对
  ├ 材料制作阶段
  │   └ PPT 内容制作
  └ 汇报组织阶段
        ├ 场地预定
        └ 领导邀请确认
```

## 技术架构

> 前后端分离架构，成熟稳定的技术组合，确保高性能、高可靠性与良好的扩展能力。

| 技术组件 | 说明 |
|----------|------|
| 🐦 **Flutter 3** | 跨平台 UI 框架，一套代码同时构建 iOS、Android 应用 |
| 🍃 **Spring Boot** | 企业级 Java 后端框架，高性能 RESTful API 服务 |
| 🧠 **DeepSeek AI** | 领先的中文大语言模型，驱动智能任务分解与整理 |
| ⚡ **SSE 实时推送** | Server-Sent Events，低延迟实时消息推送技术 |
| 🗄️ **MyBatis** | 灵活高效的持久层框架，精准的 SQL 控制与映射 |
| 🔒 **SHA-256 安全** | 密码散列加密存储，Token 鉴权，保障账户安全 |
